# asteroids

Arcade classic Asteroids, coded in Python with Pygame!

1. Create the venv
  ```
  python3 -m .venv
  ```

2. Activate the venv
  ```
  source .venv/bin/activate
  ```

3. Install the requirements.txt
  ```
  pip install -r requirements.txt
  ```

3. Run the main program
  ```
  python3 main.py
  ```

Enjoy!
